## DESCRIPCION
Este programa está hecho en Java y sirve para ordenar un arreglo de números usando dos métodos: inserción y shell.El usuario puede elegir si quiere ordenar de forma ascendente o descendente.También se muestran los pasos mientras se ordena el arreglo, junto con el número de comparaciones, cambios e iteraciones.

## CONCLUSION
Con este programa pude entender mejor cómo funcionan los métodos de inserción y shell, ya que se puede ver paso a paso cómo se van ordenando los números. También noté que, aunque ambos sirven para lo mismo, trabajan de forma diferente y eso afecta la cantidad de comparaciones y movimientos que hacen. En general, me ayudó a practicar la lógica de programación y a comprender mejor cómo funcionan los algoritmos de ordenamiento en la práctica.
